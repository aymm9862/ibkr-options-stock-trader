## ⚙️ Initial system requirements

Your computer needs to meet these basic standards to run the application well:

* Windows 10 or Windows 11.
* A stable internet connection.
* Interactive Brokers Trader Workstation (TWS) or IB Gateway installed on your machine.
* At least 4 gigabytes of RAM.

## 🚀 Setting up the software

Follow these steps to launch the application for the first time:

1. Locate the downloaded `.exe` file on your computer.
2. Double-click the file to start the installation. If Windows shows a security warning, click "More info" and then "Run anyway."
3. Follow the prompts on your screen to complete the setup process.
4. Open your Interactive Brokers Trader Workstation (TWS) software and log in.
5. In TWS, go to the "Edit" menu, select "Global Configuration," and then "API" and "Settings."
6. Make sure the "Enable ActiveX and Socket Clients" box is checked. 
7. Note the "Socket port" number shown in these settings. You will need this for the next step.
8. Start the ibkr-options-stock-trader application.
9. Enter the port number you found in TWS into the settings menu of this application.
10. Click the "Connect" button to link the application to your Interactive Brokers account.

## 🖥️ Using the interface

The main window displays your account balance and current positions. You can use the search bar to look up symbols for stocks or options. 

### Placing a trade
To place an order, select a symbol and choose the "Buy" or "Sell" button. A window will appear asking for the quantity and the type of order you want to place. Enter these details and click "Submit." The application sends this request to your broker instantly.

### Viewing your options
Use the options tab to see a list of contracts. Click on the contract you want to trade to see its details. The interface updates the price information as the market moves. 

## 🛠️ Common troubleshooting steps

If you cannot connect, check these items:

* Verify that TWS is open and you are logged into your account.
* Check that the port number in the application matches the "Socket port" setting in TWS.
* Ensure you have an active internet connection.
* Restart both TWS and the application if the connection light stays red.

## 🛡️ Security and account safety

This software runs locally on your machine. Your login information stays within the Interactive Brokers software. This application only acts as a remote control for the platform you already use. It does not store your passwords or financial data on external servers.
